## Installation

# From Source
```bash
(venv)$ git clone https://github.gatech.edu/mpatel636/SSA_Validation.git  # clone the source code

(venv)$ cd SSA_Validation

(venv)$ pip install . # install into virtual environment using pip

```
# Conda
```bash
conda install malavp::valpy
```
