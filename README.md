# Installation

## From Source
```bash
(venv)$ git clone https://github.com/Malav-P/valpy.git  # clone the source code

(venv)$ cd valpy

(venv)$ pip install . # install into virtual environment using pip

```
## Conda
```bash
conda install malavp::valpy
```
